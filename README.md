# ESTA 软件组 2026 培训仓库

本仓库用于哈尔滨工程大学电子科技协会（ESTA）2026届软件组成员的培训任务提交。所有作业通过 Pull Request 提交，不接受其他形式。

## 参与方式

1. Fork 本仓库到你的 GitHub 账号
2. 在 `submissions/` 下创建以**你的姓名拼音**命名的文件夹
3. 按任务说明完成内容，整理好目录结构
4. 发起 Pull Request，标题格式：`[taskXX] 姓名`

## 任务列表

|------|------|------|
| [task01](./tasks.md#task01--提交第二次合作任务项目工程) | 提交第二次合作任务项目工程 | 进行中 |

## 目录结构

每位成员在此目录下创建以**姓名拼音**命名的文件夹，任务提交放在对应子目录中。
```
submissions/
└── 你的姓名/
    ├── task01/
    |    ├── README.md
    |    └──your-project/
    └── task02/...
```

## 注意事项

- PR 未通过 checklist 不予合并
- 截止时间后提交的 PR 不再 review
- 遇到问题优先自行排查，其次查文档，最后再问人
