# 任务列表

## task01 — 提交第二次合作任务项目工程

### 任务目标

将你们在第二次合作任务中完成的项目工程上传到 GitHub，并提交至本培训仓库。

### 任务内容

#### 第一步：将项目上传到 GitHub

在自己的 GitHub 账号下创建一个新仓库，将第二次合作任务项目工程推送上去：

#### 第二步：向本仓库提交作业

1. Fork 本仓库到你自己的 GitHub 账号并clone到本地
2. 在 `submissions/` 下创建以**你的姓名拼音**命名的文件夹,在此文件夹下新建文件夹task01/
3. 在该文件夹内将你上传到github的第二次合作任务项目工程clone进来
4. 在该文件夹内提交一个 `README.md`，内容包含：
   - 你的项目链接（第一步创建的 GitHub 仓库地址）
   - 项目简要说明

### 目录结构

```
submissions/<姓名拼音>/task01/
├── README.md
└──your-project/
```

### PR 要求

- 标题格式：`[task01] 姓名`
- PR 内容请参照 [PR 模板](.github/pull_request_template.md) 填写
