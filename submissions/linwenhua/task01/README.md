# Task01 — DSO 双模示波器

## 项目仓库

https://github.com/Solstice242/Refresh_Group

## 简要说明

基于 STM32H750 的数字存储示波器（DSO），核心功能：

- **双模测频**：测周法（TIM2 输入捕获）+ 测频法（TIM8 边沿计数），自动切换阈值 3.3kHz，全程精度 < 0.33%
- **FFT 频谱分析**：1024 点实数 FFT，Hanning 窗，柱状图 + 前 5 次谐波显示
- **自动增益适配**：8 挡滞回切换，死区率 35%，防抖动
- **菜单系统**：TIM3/TIM4 双编码器驱动，函数指针回调，替代独立按键
- **自适应滤波**：三点中值去毛刺 + EMA 平滑降噪
- **自校正**：TIM12 PWM 内部校准信号，零点 + 幅值 + 频率三合一校正
